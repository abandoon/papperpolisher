# Introduction Writing Policy

## Core Strategy: Problem -> Limitation -> Insight -> Method

### 1. Problem Framing
- Start with the specific technical challenge, not broad industry context.
- Keep background narration minimal.

### 2. Limitation Identification
- Clearly articulate why existing work fails or is insufficient.
- Be precise about the "gap" being addressed.

### 3. Insight-driven Motivation
- Explain the *why* before the *how*.
- Compress long motivation paragraphs into sharp, insight-focused sentences.

### 4. Method Summary
- State the contribution clearly.
- Avoid listing every minor implementation detail here.
